# Phishing Email Investigation – Case 01

## Objective

This project documents my investigation of a suspicious email to determine whether it is phishing, malicious, or legitimate.

## Skills Demonstrated

* Email header analysis
* Phishing indicator identification
* URL and domain analysis
* IOC extraction
* VirusTotal threat intelligence checks
* Incident reporting

## Investigation Process

1. Review the sender email address and subject line.
2. Identify suspicious language, urgency, links, and attachments.
3. Analyze email headers, including SPF, DKIM, DMARC, and Received headers.
4. Extract Indicators of Compromise (IOCs).
5. Check suspicious domains, URLs, IP addresses, or hashes using VirusTotal.
6. Document findings and provide a final verdict with recommended actions.

## Project Files

* `report.md` — Full phishing investigation report
* `iocs.txt` — Extracted indicators of compromise
* `screenshots/` — Supporting evidence and analysis screenshots
