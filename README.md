# Phishing Email Investigation – Case 01

A beginner SOC analyst project focused on phishing email investigation, email header analysis, IOC extraction, and incident reporting.

## Project Overview
This project analyzes a suspicious phishing email that impersonates a reputable organization. The investigation includes reviewing the email body, raw email source, sender details, attachment information, and technical indicators.

## Skills Demonstrated
- Phishing email analysis
- Email header analysis
- IOC extraction
- Raw email source investigation
- Attachment review
- Incident reporting

## Files Included
- `report.md` — Full phishing investigation report
- `iocs.txt` — Extracted indicators of compromise
- `screenshots/` — Supporting screenshots from the analysis

## Final Verdict
The email was identified as a phishing attempt.

## Note
All suspicious indicators are documented in a safe/defanged format where needed.
